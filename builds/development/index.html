About This Site

Hi, I created this site as a both a resume and a demonstration of the features of bootstrap 3. Please read on, and let me know what you think.
Norman Quates

I put more items in this site than I would normally put in a resume so readers can see how a simple one page site can show summary information, and also quickly show detailed information on any aspect of interest.

Everything in this site is as accurate as I can ascertain. Please send me an email on LinkedIn with comments as this is my first resume since 1999. I have this on GitHub: https://github.com/nquates/resume-using-bootstrap3 as well and would welcome comments on my web techniques.

This site is very responsive to different browser environments and was created with a mobile first strategy. I will be making another similar site incorporating angularJS and will add a link when that site is ready.

I would like to acknowledge the excellent training videos at Lynda.com that I used as a refresher and to supplement what I picked up on my own about bootstrap 3. In particular, courses by Ray Villalobos: Bootstrap 3 Essential Training  and  Web Project Workflows with Guip.js, Git and Browserify.

Another mobile friendly site that I recently created is the Edata Weather Site. I used mainly CSS3 Flex Boxes for the Edata Weather Site. On this site, I used graphics and some Telerik tools to read live JSON data files that I create from data gathered from weather equipment located on the premises. The gauges and readings are updated in two minute intervals. The radar images are obtained from a program called Storm Predator. I am using my own technique to animate the radar images to remove problem frames in Storm Predator's animation. This weather site was also done with a mobile first strategy and looks great on a phone.

The Edata Web site: www.edaxc.com was done quite a few years back. It has a lot of "neat gadgets" using jQueryUI. This site was not done with a mobile first strategy and therefore is not quite as accurate when viewed on a mobile phone.
